## 🚀 Getting Started

### Prerequisites

- Node.js installed on your machine

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/randQuote.git
cd markdown-previwer
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```
